## <u>Teamate</u> Project Background
- ### 필요성
  - 기존의 보행 보조 도구(흰 지팡이)는 움직이는 물체나 접근 방향을 판단 할 수 없음
  - 때문에 보행시 돌발 상황에 대처하기 어려움
 
- ### 기존 해결책의 문제점
  - 물체 종류 파악 불가 : 지팡이로 판단하는 장애물은 물체의 종류나 형태를 판단하기 어려움
  - 예) 사람, 전봇대, 자동차 --> 모두 장애물
    
  - 시각 정보 인식 불가 : 시각장애인이 목적지를 파악할 때 주변의 간판, 표지판 등의 시각 정보의 도움을 받을 수 없음
  
## System Design
  - ### System Requirements
    - 객체 탐지 및 객체 상태 추정 가능
    - 시각장애인 사용을 고려한 음성 안내 제공
    - 사용자에게 보행 방향을 소리로 안내
    
## Case Study
  - ### Description
  - 보행 중 정적/동적 장애물 인식 
  - 간판 텍스트 인식
  - 다양한 실험 환경(학교 복도, 주차장)
  
  
## Conclusion
  - ### 시각장애인을 위한 지능형 보행 보조 시스템을 구현
  - ### YOLO + RGB-D 카메라 기반 객체 탐지
  - ### 탐지된 객체를 바탕으로 보행 방향 안내
  - ### OCR을 통해 간판 인식 및 음성 안내
